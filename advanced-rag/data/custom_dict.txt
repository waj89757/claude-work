# ================================================
# 自定义词典 - 每行一个词
# ================================================
# 
# 格式很简单：
#   词语 词频 词性
#
# 你只需要关心第一列"词语"，后面两列可以固定写 10000
#
# ================================================

# --- 举例：这些词如果不加词典，会被切碎 ---

# 加了词典: "向量数据库" → ["向量数据库"] ✓
